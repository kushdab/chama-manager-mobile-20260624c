# Chama Manager Mobile 2026

A TypeScript-based core engine for managing Kenyan Investment Groups (Chamas). This system handles member registration, contributions, and loan processing with interest calculation.

## Features
- **Member Management**: Track individual savings and debt.
- **Contribution Tracking**: Real-time updates to the group kitty.
- **Loan Logic**: Implements a 3x multiplier limit and 10% interest rate.
- **Financial Reporting**: Summary of group assets and member standing.

## Installation
1. Ensure you have Node.js installed.
2. Install dependencies:
   ```bash
   npm install
   ```

## Usage
Run the simulation script:
```bash
npm start
```
